{% trans 'Medlemmer' %}

Medlemmer er siden hvor du kan registrere nye medlemmer, redigere gjellende medlemmer eller slette medlemmer. Du kan også søke etter medlemmer ved eventuelle arrangementer.

{% trans 'Statestikk' %}

Statestikk siden vil kunne gi deg verdifull statestikk over din medlemsbase. Kjønnsfordeling eller medlemsregistreringer fra måned til måned er noe av hva du kan finne her.

{% trans 'Administrator' %}

I administrasjon siden kan du legge til nye administratorer, redigere disse eller slette eksisterende administratorer. Dette er hvem som har tilgang til siden.

{% trans 'Kalender' %}

Her kan du registrere interne og vanlige arrangemneter. Disse vil hovedsaklig bli synlig for administratorer av denne siden. Du kan også synce med google kallender.

Arena

Mail
